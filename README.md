## なにこれ
Lazy K (https://tromp.github.io/cl/lazy-k.html) と同じく,
SKI コンビネータ計算に基づくプログラミング言語です.
Lazy K (Unlambda 記法) との対応は,
```
麺   = s
汁   = k
ブタ = i
xy！ = `xy (関数適用, ただし後置)
```
となっていて, これ以外の文字はコメントとして扱われます.
